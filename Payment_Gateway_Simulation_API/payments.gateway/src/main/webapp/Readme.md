# Payment Gateway Simulation API

A backend-focused project simulating real-world payment gateway behavior.

## Features
- JWT Authentication
- Card validation using Luhn algorithm
- Payment lifecycle: SUCCESS / FAILED / PENDING
- Refund processing
- Transaction history
- MySQL persistence

## Tech Stack
Java, Spring Boot, Spring Security, JWT, MySQL, REST APIs

## How to Run
1. Configure MySQL in `application.yml`
2. Run Spring Boot app
3. Test APIs using Postman
4. Optional: open frontend/index.html

## Why This Project
Designed to mirror real payment systems used in fintech and banking domains.
